# IT Ticket Priority System

Simple Python project simulating IT support ticket classification system.

## Features
- Generates random IT issues
- Assigns priority levels based on severity
- Logs all tickets to file
- Generates summary report

## Tech Stack
- Python
- File handling
- Basic decision logic
- Data analysis

## How to run
1. Run main.py (generate tickets)
2. Run analyzer script (optional report)

## Example output
Ticket 1234 | Server down | Severity 9 → CRITICAL
